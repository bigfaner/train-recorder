# Task 2.4: Core Use Cases — WeightSuggester & ScheduleCalculator

## Description

Implement the two core use cases that drive the app's intelligent features.

### WeightSuggester
Calculate the next suggested weight based on recent session history for an exercise:
- **All sets completed** → increment by the exercise's default increment
- **Partial fail** → hold current weight (no change)
- **2 consecutive fails** → deload 10%, round down to nearest increment
- **3 consecutive passes** → return `CONSIDER_MORE` hint (user may want a bigger increment)
- **First time (no history)** → return `null` (no suggestion available)

### ScheduleCalculator
Compute the training schedule from plan configuration:
- **weekly_fixed**: repeat training days each week on fixed days
- **fixed_interval**: cycle through training days with configured rest days between
- Overlay completed workout sessions onto the schedule
- Track skipped days and consecutive skip streaks

### Unit Tests
Include comprehensive unit tests for all edge cases of both use cases.
