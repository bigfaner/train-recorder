## Description

### Onboarding (US-18)
Implement first-use onboarding: welcome screen, plan template selection (PPL / Upper-Lower / Full Body), exercise pre-fill, completion.

### Data Export
Generate JSON file from all tables, date range filter, share via platform share sheet.

### Data Import
Validate JSON schema, regenerate IDs, merge with existing data, transaction-based with rollback on failure.

### Clear All Data
Confirmation dialog, delete all training data, keep exercise library and settings.
