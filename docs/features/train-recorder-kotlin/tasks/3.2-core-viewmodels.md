## Description

Implement BaseViewModel<S, E> abstract class and core ViewModels:

- **CalendarViewModel**: Loads schedule via ScheduleCalculator, handles month change, date select, skip/unskip/drag, consecutive skip tracking.
- **WorkoutViewModel**: Manages workout session state, records sets, handles timer, reorder/skip exercises, partial complete.
- **PlanViewModel**: Loads plans, create/update/activate/delete.

Each ViewModel uses `StateFlow<UiState>` and sealed Event classes per tech-design Presentation Layer section.
