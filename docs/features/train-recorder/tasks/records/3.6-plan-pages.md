### Key Decisions

- plan-helpers.ts contains all pure logic (validation, SetsConfig parsing, formatting) as testable functions separate from UI
- Plan editor supports both weekly_fixed (weekday toggle + type assignment) and fixed_interval (rest days stepper + day order list) schedule modes

- SetsConfig discriminated union (fixed/custom) parsed/serialized via helper functions, matching schema JSON format
- Inline exercise picker in training day editor (simpler than full modal for MVP; modal can be added later)
- validatePlan returns both errors (blocking) and warnings (non-blocking, e.g. 7 training days with no rest)
