# Task 2.1: Exercise & Settings Repository Implementations

## Description

Implement `ExerciseRepository` and `SettingsRepository`.

### ExerciseRepository
- CRUD operations for exercises (create, read, update, delete)
- Search exercises by name
- Filter exercises by category
- `seedDefaultExercises()` — inserts the 19+ built-in exercises from PRD section 5.5 on first launch:
  - **core**: 深蹲 5kg / 卧推 2.5kg / 硬拉 5kg / 推举 2.5kg
  - **upper_push**: 上斜卧推 2.5kg / 哑铃卧推 2.5kg / 双杠臂屈伸 2.5kg
  - **upper_pull**: 杠铃划船 2.5kg / 引体向上 2.5kg / 高位下拉 2.5kg / 哑铃划船 2.5kg
  - **lower**: 前蹲 2.5kg / 腿举 5kg / 罗马尼亚硬拉 2.5kg / 腿弯举 2.5kg
  - **abs_core**: 卷腹 2.5kg / 平板支撑 5kg / 健腹轮 2.5kg
  - **shoulder**: 侧平举 1kg / 面拉 1kg / 推举哑铃 2.5kg

### SettingsRepository
- Get and update user settings
- Unit conversion logic (1 kg = 2.20462 lb)
- Export/import data
- `clearAllData()` — wipe all user data

### SQLDelight Query Files
- Create `.sq` query files for both repositories with all necessary queries.
