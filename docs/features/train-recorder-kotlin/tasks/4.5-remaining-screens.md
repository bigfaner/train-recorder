## Description

Build remaining screens:

- **Feeling screen** (UF-8, push after workout): Fatigue/satisfaction sliders, per-exercise notes, save/skip.
- **Body data screen** (UF-6, Tab 4): Latest data card, record form, trend chart.
- **Other sport screen** (UF-7, push from calendar): Sport type grid, dynamic metric inputs, custom sport creation.
- **Exercise library screen** (UF-4, push from settings): Category list, search, detail view, CRUD.
- **Settings screen** (UF-10, Tab 5): Grouped settings list, unit toggle, export/import, clear data.
