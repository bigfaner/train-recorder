# Task 2.3: Supporting Repository Implementations

## Description

Implement the five supporting repositories.

### BodyDataRepository
- CRUD for body measurements
- Date-range queries for historical data

### OtherSportRepository
- Sport type CRUD with metric configuration
- Record creation with metric values

### FeelingRepository
- Save and update feeling entries
- Support exercise-level notes within a session

### PersonalRecordRepository
- Track max weight and max volume per exercise
- Auto-update records after workout completion
- Recalculate records when a workout session is deleted

### WeightSuggestionRepository
- Cache layer for weight suggestions
- Recalculate suggestions from `exercise_set` history

### SQLDelight Query Files
- Create `.sq` query files for all five repositories.
