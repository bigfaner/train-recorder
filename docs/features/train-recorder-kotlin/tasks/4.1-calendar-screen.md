## Description

Build Calendar screen (UF-1, Tab 1). Month view grid with training type color bars (push=blue, pull=green, legs=orange). Date cells show status markers (completed checkmark, skipped dashed). Bottom card shows today summary with "Start Workout" button. Month navigation with swipe. Training type filter chips. Empty state when no plan. Follow `ui-design.md` Component 1 layout and Apple Design Language.
