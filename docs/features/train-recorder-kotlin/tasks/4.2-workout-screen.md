## Description

Build Workout execution screen (UF-2, push page, hides tab bar). Exercise card list with current exercise expanded. Weight input with suggested value pre-filled, reps input, "Complete Set" button. Timer panel slides up after each set. Swipe-to-skip exercise, long-press-to-reorder. Progress indicator in nav bar. Exit confirmation dialog. Follow `ui-design.md` Component 2.

## Reference Files

- `ui/ui-design.md` (Component 2: 训练执行)
- `ui/prototype/workout.html`
- `design/tech-design.md` (WorkoutScreen state)

## Acceptance Criteria

- [ ] Exercise cards expand/collapse correctly with smooth animation
- [ ] Suggested weight is pre-filled from previous workout data
- [ ] Timer panel slides up after set completion
- [ ] Swipe-to-skip exercise works with confirmation
- [ ] Long-press-to-reorder works with drag handle
- [ ] Exit confirmation dialog shows completed exercise count
- [ ] Progress indicator in nav bar reflects workout progress
- [ ] Recording a set takes <=2 clicks (pre-filled weight + tap complete)
- [ ] Tab bar is hidden during workout
