## Description

Implement TimerService for Android (ForegroundService + LocalNotification) and iOS (BackgroundTask + LocalNotification). Timer is timestamp-based: `start_timestamp + total_duration_seconds` persisted in TimerState table. On app resume, calculate `remaining = start + duration - now`. If `remaining <= 0`, trigger alert.

### Platform Details

- **Android**: ForegroundService with notification showing countdown, survives app kill.
- **iOS**: Background task with local notification.
- Cancel timer on workout exit.
