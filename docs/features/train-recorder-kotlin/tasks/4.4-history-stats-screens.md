## Description

Build History screen (UF-5, Tab 3) with 4-segment tabs:

1. **History**: Session cards list.
2. **Progress**: Exercise selector + line chart + PR markers.
3. **Volume**: Bar chart + summary card.
4. **PR**: Exercise PR cards.

Build Stats screen (UF-9, Tab 4):

- Hero card (weekly volume + change%).
- 4-grid summary.
- Weekly volume bar chart (8 weeks).
- PR list (1RM estimates).
- Training frequency heatmap (4 weeks).

Use custom Canvas chart components from task 3.4.
