# Task Record: 2.5 Unit Conversion + Data Export/Import Services

## Summary

Implemented three utility services: UnitConversion (kg/lbs conversion with barbell plate rounding), DataExport (JSON export of all training data with date range filtering), and DataImport (schema validation, conflict detection, and merge with skip/overwrite strategies). All services use dependency injection for testability.

### Key Decisions

- roundToPlate generates all achievable sums from standard plates (each 0-2 copies) rather than requiring a barbell, making it work for dumbbells and lighter weights too
- DataExport and DataImport use adapter interfaces (FileSystemAdapter, ShareAdapter, FileReaderAdapter) for dependency injection, enabling testing without native expo modules
- Import wraps all database writes in a transaction with rollback on error
- Export converts bigint biz_key values to number for JSON serialization
- Conflict detection checks biz_key existence before import, supporting both skip and overwrite strategies
