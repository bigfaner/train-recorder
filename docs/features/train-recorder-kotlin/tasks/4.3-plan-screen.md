## Description

Build Plan management screen (UF-3, Tab 2). View mode: active plan card with training day cards showing exercises. Edit/Create mode: plan name, plan mode toggle, schedule mode toggle, day-of-week buttons or interval stepper, training day editor with exercise list. Exercise picker modal from exercise library. Fixed mode (uniform sets/reps/weight) and Custom mode (per-set config). Follow `ui-design.md` Component 3.

## Acceptance Criteria

- [ ] View/edit modes switch correctly with smooth transition
- [ ] Training day cards show exercise details (name, sets, reps, weight)
- [ ] Exercise picker opens with multi-select from library
- [ ] Fixed mode inputs render uniform sets/reps/weight fields
- [ ] Custom mode inputs render per-set configuration fields
- [ ] Plan saves successfully and appears in calendar
- [ ] Schedule mode toggle switches between fixed days and interval
- [ ] Day-of-week buttons and interval stepper work correctly
