## Description

Implement remaining ViewModels:

- **HistoryViewModel**: 4 tabs — history list, progress chart data, volume data, PR list.
- **StatsViewModel**: Weekly/monthly stats, 1RM estimation, heatmap data.
- **FeelingViewModel**: Fatigue/satisfaction sliders, exercise notes, high-fatigue warning.
- **SettingsViewModel**: Unit toggle, rest time, export/import, clear data.
- **BodyDataViewModel**: Measurements CRUD, trend data.
- **OtherSportViewModel**: Sport types, dynamic metrics, create custom.
- **ExerciseLibraryViewModel**: Browse, search, filter, select mode, CRUD.
