## Description

Set up Compose Multiplatform Navigation with type-safe routes (all 15 routes from tech-design). Create tab bar scaffold (5 tabs: Calendar, Plan, History, BodyData, Settings). Implement custom Compose Canvas chart components:

- **LineChart**: Progress curves with PR markers.
- **BarChart**: Volume trends.

Charts support touch-to-inspect data points.
